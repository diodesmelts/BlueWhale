API Endpoints

Blue Whale Competitions offers a robust API for accessing competition data and more. Here are the available endpoints:

Status GET

Check the API status and version information.

/api/status

Competitions GET

Get a list of all available competitions.

/api/competitions

Leaderboard GET

Get the current user leaderboard.

/api/leaderboard

User Stats GET

Get statistics for the current user.

/api/user-stats

Logo GET

Get the application logo URL.

/api/settings-logo

Banner GET

Get the application banner URL.

/api/settings-banner
